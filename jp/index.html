静的サイトを構築する
使いやすいフレームワーク

Middleman はモダンな web 開発のショートカットやツールを 採用した静的サイトジェネレータです

          $ gem install middleman
        
              $ middleman init project

インストール

gem をインストールし, 新たに Middleman を使ったサイトを作成するために CLI から middleman init my_project を実行します。

インストール方法を参照する

サイトのカスタマイズ

cd my_project してください。 Middleman は細かい設定なしに動作します。各ディレクトリの意味や サイトのカスタマイズの行い方を理解するためにドキュメントを読んでください。

カスタマイズ方法を参照する

ビルドとデプロイ

bundle exec middleman build を実行するだけで, プロダクション用のサイトデータを出力できます。 オープンソースのデプロイツールを利用することでサイトを公開できます。

ビルド方法の説明を参照する

オープンソースコミュニティ

Middleman は活発なコミュニティに支えられています。あなたに質問がある時, オープンソースの機能に貢献したい時, コミュニティページが 正しい場所にあなたを導くでしょう。
